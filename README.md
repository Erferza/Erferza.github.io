# Ferza Er Portfolyo

Staj başvurularında kullanılmak üzere hazırlanmış tek sayfalık kişisel portfolyo sitesi.

## Yerelde Açma

`index.html` dosyasını tarayıcıda açmak yeterlidir.

## GitHub Pages ile Yayınlama

1. GitHub'da `Erferza.github.io` adında public repo oluştur.
2. Bu klasördeki dosyaları repoya yükle.
3. Repo `Settings > Pages` ekranında source olarak `main` branch ve `/root` seç.
4. Site birkaç dakika içinde `https://Erferza.github.io` adresinde yayına çıkar.

Not: Kodları paylaşılamayan projeler portfolyoda araştırma/case-study olarak anlatıldı. Bu yaklaşım, özel repo veya TÜBİTAK kapsamındaki paylaşım kısıtlarını ihlal etmeden projeyi görünür kılar.
